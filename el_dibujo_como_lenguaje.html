El Dibujo como lenguaje

El dibujo es un lenguaje visual universal que comunica ideas, emociones y conceptos mas allá de las palabras, funcionando como un sistema de emisor (dibujante), receptor (observador), mensaje (contenido) y código (estilo, símbolos).
